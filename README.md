📌 项目简介

Name Dance 6 AI Ultimate 是一款基于 Web 前端技术实现的高可用智能点名系统。
该系统具有 抽取公平、界面精美、特效酷炫、数据可保存、可分析 等特点。
你只需打开 index.html，即可在任何浏览器中使用完整功能，无需服务器，无需部署。

本项目特别适用于：

课堂点名

小组活动抽签

会议随机发言

直播互动抽奖

随机测试/问答

任何需要“公平随机”的场景

系统包含大量现代化功能，如 AI 算法调控、权重系统、概率实时图表、历史分析、分组加载、文件范围选择等，是当前类点名软件中功能极为全面的一款。

✨ 功能特点
1. 智能点名（可多选）

支持一次抽取 1～N 个姓名

动态滚动动画

抽取到的姓名可自动从可用名单中移除，避免重复

2. 权重与公平算法

系统支持多维度的抽取调优：

项	说明
自定义权重	可为任意学生设置抽取概率权重
AI 公平抑制	高频被抽取者会自动降低权重
抽样抖动	随机扰动权重，避免出现“确定性偏差”
温度 (Temperature)	调整随机性整体强度
预洗牌 (Shuffle)	提高随机分布均匀性
3. 概率统计（即时计算）

提供两种可切换展示方式：

柱状图模式（Chart.js）

精简概率列表模式

根据实际权重、历史抽取次数、AI 公平系数动态计算实时概率。

4. 历史记录系统（长期保存）

全自动保存每次抽取记录（时间/范围/抽取人数/结果）

可 删除个别记录 / 清空所有记录

导出 CSV

导入 CSV（含校验自动解析）

专用 历史分析面板（Top10 抽取排行榜、总次数统计）

5. 分组与范围管理（支持外部 JS 文件）

支持加载多个班级文件
